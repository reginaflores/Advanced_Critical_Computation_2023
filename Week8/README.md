### Git when you are starting a new repo:

	git init
	git add README.md
	git branch -M main
	git remote add origin https://github.com/<user name>/<repo name>
	git add -A 
	git status
	git commit -m "first commit" 
	git push -u origin main


### Git when you have an existing repo:

	git add -A  
	git status    
	git commit -m "comment"
	git push -u origin main
